I dette tema opsatte jeg selvstændigt mit første website ud fra layoutdiagrammer, wireframes og indhold, som blev udleveret af vores undervisere.

Temaet gav mig grundlæggende forståelse for HTML, CSS og metoder til udvikling og implementering af responsive brugergrænseflader, der reagerer på interaktioner og forskellige skærmdimensioner.

Undervejs blev jeg også introduceret for værktøjerne Visual Studio Code, Figma og Filezilla, og lærte mere om designkonventioner og gestaltprincipper.
